# jieyuc-ai-tools
## 实现目的
项目主要用来整合各类大预言模型的接入, 实现一套SDK接入所有主流模型, 主要用作后续项目中快速使用各类模型, 完成对应的工作, 避免重复遭轮子, 
开源社区中虽然也存在类似的工具, 但处于学习的目的, 还是决定自己实现一套ai工具包, 来适应自己的编程习惯, 方便后续项目中直接使用.
最初阶段仅仅实现一套SDK可以快速使用, 后续维护中可能会加入http api, rpc调用和ctl-tools的实现.
